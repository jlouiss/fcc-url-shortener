URL Shortener Microservice

How this API works:

POST /api/shorturl/new - { "url": "https://www.google.com" }

returns { "original_url": "https://www.google.com", "short_url": 1 }


GET /api/shorturl/:number redirects to matching URL

Example

POST /api/shorturl/new - { "url": "https://www.twitter.com" }
(response: { "original_url": "https://www.twitter.com", "short_url": 2 })

GET /api/shorturl/2 redirects to twitter.com
